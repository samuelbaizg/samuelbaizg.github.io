文本分类概述

发表于 2020-06-25 | 分类于 NLP

  • 文本预处理

文本预处理方法

移除停用词

停用词，也叫Stop Words，通常停用词是指对于文本理解没有什么帮助的词，而且某些情况下还会引起理解问题，它通常分为两类，一类是功能词，一类是词汇词。

  • 功能词， 与其它词相比，它没有什么实际含义，比如the、is、at、which、它、的等等；
  • 词汇词，比如want，想，不如，不是等等。

神经网络概述

发表于 2020-06-21 | 分类于 深度学习

计算机神经网络包含三要素，神经元、激活函数和网络结构。

神经元

神经元由三部分组成：连接权、求和公式和激活函数；

激活函数

激活函数类似生物神经元的突触，负责将神经元的信号输出到下一个神经元，它有如下特点：

  • 一般都是非线性函数，因为非线性函数才能将神经元信号转换成非线性输出，非线性输出才能以任意精度逼近任意非线性函数，从而拟合现实世界中的非线性问题；
  • 激活函数一般是可微分函数、因为只有可微分函数才能算梯度
  • 如果激活函数一般是单调的，则误差函数凸函数，易于优化；

激活函数的选择方法：

  • 选择非线性函数
  • 选择可微连续函数
  • 选择求导计算量小的函数
  • 选择输出值域范围，避免出现恒正或者恒负的情况
  • 是否存在饱和度区间，如果有饱和度区间，使用梯度下降法求解时容易出现梯度消失的问题

常用的激活函数包括：

  • Sigmoid函数 - $f(x) = 1 / (1 + e^{-x}))$

    它的函数和导函数图形如下：

    它有如下特点：

    • 是可导连续函数，导数可以通过自身求得，求导是指数运算，计算量大；

    • 输出值域值在0 ～1之间， 当x>5或x<-5时，函数近乎于饱和，梯度接近零，在用梯度下降法求解时，会出现梯度消失；

    • 因为值域始终大于零，隐藏层加权求和后更容易落到饱和区，进而出现梯度消失；

  • Tanh函数 - $ f(x) = (e^{x} - e^{-x})/(e^{x} + e^{-x})$

    它的函数和导函数图像如下：

    它有如下特点：

    • 它是连续可导函数，求导是指数运算，计算量大；
    • 输出值域值-1～1之间，当x<-2.5或者x>2.5，函数近乎于饱和，梯度接近零，在用梯度下降法求解时，会出现梯度消失；
    • 与Sigmoid不同的是，它是零均值函数，输出不会出现恒正或者恒负的情况，它比Sigmoid陡峭，下降速度快；

  • Relu函数 - $f(x) = max(0, x)$

    它有如下特点：

    • 分段可导函数，线性函数，导数是常数，计算量小；
    • 没有饱和区，作为激活函数，没有梯度消失问题；
    • 当输入为负数时，没有梯度，输出始终为零，神经元会变成死神经元，导致矩阵稀疏；
